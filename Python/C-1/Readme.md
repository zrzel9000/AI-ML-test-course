----------------Number Guessing Game---------------------

A random number will be generated from 0 to 10
You will get 3 chances to guess the number
if you guess the number correct or if all your chances are finished then you will be asked if you want to play again
you can enter y for yes or any other number for no 
if you enter y then the game will run again
else the game will stop

Steps:
 1. Enter your guess from 0 to 10 (total chances - 3)
 2. Enter whether you want to play again
